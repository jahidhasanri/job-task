# Ecommerce Frontend Application

A modern, pixel-perfect, and fully responsive ecommerce frontend application built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **shadcn/ui**, following the provided Figma design.

This project was developed as part of a frontend assignment to demonstrate clean UI implementation, responsiveness, and best frontend development practices.
